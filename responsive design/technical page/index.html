Introduction

This documentation covers the basics of our programming language.

It is designed for both beginners and experienced developers.

console.log("Hello, world!");
  • Easy to learn
  • Lightweight
  • Versatile
Getting Started

First, install the runtime environment.

Next, create your first file and run it using the CLI.

npm install -g mylanguage

Save your file with a .ml extension.

mylanguage run hello.ml
Syntax

The syntax is simple and clean.

Functions are declared using the `function` keyword.

function greet(name) { return "Hello, " + name; }
  • Keywords are case-sensitive
  • Indentation is optional

Best Practices

Write clean and maintainable code.

Always comment on complex logic.

// This is a comment explaining the loop

Follow naming conventions and use consistent formatting.

  • Use camelCase for variables
  • Keep functions short
  • Test your code regularly
